Standard Operating Procedure

Guest Folio Charging via POS
Department: F&B / Reception
Effective: 7 April 2026
Version: 2.0

1. Purpose

This document outlines the updated procedure for charging food & beverage items to guest room folios using the SumUp POS system. Charges made on the POS are automatically posted to the guest's folio in MEWS. If a sale is voided on the POS, the charge is automatically removed from the guest's folio.

2. Regular Guest Folio Charges

F&B Team Follow these steps when a guest requests to charge to their room.

Opening the Table

  1. As soon as you open the table, tap Assign Customer on the POS.
  2. You will see a list of checked-in rooms only, displayed as:
    PET02 — Terese Moe Leiner
    PET05 — Hannah Hale
    If the guest's room is not listed, confirm with Reception that they are checked in.
  3. Select the correct room and guest name.
Room Number Format Always use the trailing zero when searching for a room. For example, type PET01 not PET1.

Ringing Up & Closing the Sale

  1. Ring up all items on the POS as normal.
  2. When the guest is ready to pay, print the bill and take it to the table for the guest to sign.
  3. Back on the POS, tap Pay.
  4. In Amount Tendered, enter the full amount including tips.
  5. Press To Guest Folio.
  6. The change shown is the tip amount — assign it as Tips.
  7. The charge will be automatically posted to the guest's folio in MEWS within seconds.

After the Sale

  1. Take the signed bill to Reception to be filed with the guest's folio records.
Tip Reminder Always add tips at the POS when closing the bill. If you forget, inform Reception so they can verify and correct it.

3. Reception Verification

Reception When receiving signed bills from the F&B team:

  1. Check the signed bill against the charge that appeared on the guest's folio in MEWS.
  2. Verify tips: If the bill includes a tip, confirm that the waiter added it correctly on the POS. The tip should appear as a separate line item on the folio.
  3. If the tip is missing or incorrect, add it manually to the guest's folio in MEWS.
  4. File the signed bill in the guest's folio folder for record-keeping.

4. Voiding a Transaction

Manager Only If a charge was posted to a guest's folio in error:

Important Do not attempt to remove or edit charges directly in MEWS. All corrections must be done through the SumUp backoffice by voiding the original sale. Go to pos.thegoodtill.com/app/sales/find_transaction and enter the transaction number shown on the MEWS folio. The system will automatically remove the charge from the guest's folio.
  1. Identify the incorrect sale on SumUp POS.
  2. Escalate to a Manager — only managers are authorised to void transactions.
  3. The manager will void the sale on the POS.
  4. The charge will be automatically removed from the guest's folio in MEWS.
  5. Verify in MEWS that the charge has been removed.

5. Event & Set Menu Charges

F&B Team For events with pre-agreed set menu pricing, follow this adjusted process.

Charging the Set Menu

  1. On the POS, select the appropriate set menu button:
    • Set Menu Breakfast
    • Set Menu Lunch
    • Set Menu Dinner
  2. Enter the agreed price for the set menu.
  3. Assign the correct number of covers.
  4. Close the bill as Guest Folio.
Do Not Assign a Room For event set menus, do not assign the bill to a guest room. The event charge is handled separately. Assigning it to a room will result in a double charge on the guest's folio.

Handling Products Served in the Set Menu

  1. Individual products served as part of the set menu must be wasted on the POS for accurate stock tracking.
  2. When wasting, select the reason: Set Menu.
Why We Waste Instead of Discount Previously, products in set menus were given a 100% discount. We now waste them with reason "Set Menu" instead. This gives us accurate stock and waste reporting without distorting discount figures.
